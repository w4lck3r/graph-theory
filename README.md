# dm-graphes-21-22-gr1

## Que faire avec ce dépôt ?

- un des membres du binôme réalise un 'fork' (et surtout pas un clone)
- il y invite l'autre membre du binôme, ainsi que son enseignant

## Organisation du dépôt

Dans le dépôt vous trouverez :
- fichier nommé binome.md qui contiendra le nom des deux membres du binôme (NOM Prénom, 1 par ligne)
- un fichier nommé rendu1/cr1.md au format Markdown qui devra contenir le compte-rendu du 1er rendu intermédiaire. Si des images sont ajoutées elles le seront dans le répertoire rendu1
- un fichier nommé rendu2/cr2.md au format Markdown qui devra contenir le compte-rendu du 2ème rendu intermédiaire. Si des images sont ajoutées elles le seront dans le répertoire rendu2
- un répertoire src avec les trois scripts à rendre et un fichier README.md à compléter
- un répertoire src/exemples avec au moins un exemple pour chacun des trois problèmes

## Echéances

- rendu 1 : mercredi 24 novembre, 22h
- rendu 2 : mercredi 8 décembre, 22h
- rendu final : vendredi 17 décembre, 22h
